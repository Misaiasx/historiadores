JavaScript

JavaScript é uma linguagem de programação interpretada, utilizada principalmente para adicionar interatividade e dinamismo às páginas web.

Criada em 1995 por Brendan Eich, inicialmente para rodar no navegador Netscape, hoje é uma das linguagens mais populares do mundo, usada no frontend e backend.

Com JavaScript, é possível manipular elementos da página, validar formulários, fazer animações, comunicação com servidores e muito mais.
