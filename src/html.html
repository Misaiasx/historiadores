HTML

HTML (HyperText Markup Language) é a linguagem padrão utilizada para estruturar e apresentar conteúdo na web.

Criada por Tim Berners-Lee em 1991, o HTML permite criar páginas web com texto, imagens, links, vídeos e outros elementos multimídia.

Com o passar dos anos, o HTML evoluiu para a versão HTML5, trazendo novas funcionalidades que permitem construir aplicações web modernas e interativas.
